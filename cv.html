Experience

Sophisticated Pedagogical Practice
http://sophisticatedpedagogicalpractice.nz/

Developed using Laravel, Sophisticated Pedagogical Practice is a website intended to help a client develop her forthcoming book by allowing her to present and revise work in progress chapters of the book, and receive comments from approved users known to the client. The site was designed to only allow comments from people given approval by the client to post comments on the site. I developed a system where a new user would be informed that they would receive an email informing them if they had been approved. By clicking an approval button next to the name of the user on the user approval page, an email would automatically be sent to the user informing them that they had been approved and could now comment on the website. The client also wanted basic word processing tools, in particular the ability to add charts. This functionality was added via tiny MCE integration, which also added the ability to easily embed YouTube videos into posts and hotlink images from around the web.

Full back end functionality was developed with Laravel, the site was designed mobile first, and is fully responsive. I’m proud of the automatic email based password reset function. The site was inspired by Scandinavian minimalist sites I was researching at the time.


Hope Springs
http://matthew.chamberlain.yoobee.net.nz/_Assignments/WE03/public_html/index.php

Hope springs is a basic web forum for survivors of sexual abuse. It is a simple site with a bootstrap frontend and a hand coded PHP backend. The challenging thing about developing the site was writing a complete CRUD enabled backend in pure PHP and finding all the bugs. This challenge taught me a lot about PHP and MySQL. This site was my first attempt at making a content management system from scratch.
